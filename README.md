# WonderfulEditor

ruby 2.7.2

```
Rails 6.0.4

## Database creation
```
$ bundle exec rails db:create
```

```
$ bundle exec rails db:migrate
```
## How to run the test suite
```
$ bundle exec rails test
```
## Execute rails server
```
$ bundle exec rails s
